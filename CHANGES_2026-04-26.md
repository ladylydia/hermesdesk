## 1. 个人微信路线 C（Python + Tauri + 壳 UI）

### 1.1 `python/src/weixin_qr_worker.py`（新）

- 与 `desktop_entrypoint` 一致：`HERMESDESK_DATA_DIR`、`HERMES_HOME`（`…/hermes-home`）、`sys.path`（`hermes` + `site-packages`）。
- `asyncio.run(gateway.platforms.weixin.qr_login(...))`；用 **保存的 `_ORIGINAL_PRINT`** 打补丁捕获扫码 URL 写入 **`weixin_qr_progress.json`**（避免把 `builtins.print` 指到仍调用 `builtins.print` 的函数 → **无限递归**）。
- 成功后调用 **`hermes_cli.config.save_env_value`**，与 CLI **`_setup_weixin`** 的安全默认对齐（`WEIXIN_DM_POLICY=pairing`、`WEIXIN_GROUP_POLICY=disabled` 等），并写 **`weixin_qr_result.json`**。

### 1.2 `python/build_bundle.ps1`

- 将 **`weixin_qr_worker.py`** 复制到 **`python/dist/runtime/`**（与 `desktop_entrypoint.py` 同级）。

### 1.3 Tauri：`tauri/src/weixin_qr.rs`（新）、`tauri/src/lib.rs`

- **`AppState.weixin_qr_child`**：`tokio::process::Child`，**`CREATE_NO_WINDOW`** 起 worker。
- 命令：**`cmd_weixin_qr_start` / `cmd_weixin_qr_status` / `cmd_weixin_qr_cancel` / `cmd_restart_embedded_hermes`**（后者包装已有 **`respawn_embedded_hermes_python`**）。
- **`RunEvent::ExitRequested`**：先 **`clone` `Arc<Mutex<…>>`** 再 `drop(State)`，并对 **`try_lock`** 使用 **中间变量**，避免 **E0597**；退出时 kill weixin 子进程。
- 子进程环境：**`env_remove("PYTHONPATH")`**，避免继承开发机 `PYTHONPATH` 覆盖 bundle 的 **`site-packages`**（见下文 §4）。

### 1.4 前端

- **`web/src/components/WeixinQrRouteCBlock.tsx`**：扫码状态机、自动/手动 **`cmd_restart_embedded_hermes`**、**`restartBusy` / `restartErr`**、**`ipcErr`**、扫码链接下 **`weixinAfterScanHint`**（说明腾讯页可无提示属正常）。
- **`web/src/advanced/Settings.tsx`**：个人微信区块改用该组件。
- **`web/src/locales/strings.ts`**：`settings.weixin*`、`weixinAfterScanHint`、`weixinRestartBusy`、`weixinRestartFailed` 等。

## 3. 设置向导：API 页「每次进来都空」

**原因**：密钥**故意不落盘**到 `sessionStorage`（`updateDraft` 持久化会剥离 `apiKey`），故密钥框始终空，看起来像未保存。

**改动**：

- **`tauri/src/secrets.rs`**：新增 **`LlmConfigPreview`**（`hasSecret` + `settings.json` 里 **`provider`** 非敏感字段）及命令 **`cmd_llm_config_preview`**；**`lib.rs`** 注册。
- **`web/src/onboarding/steps/GetAccessPass.tsx`**：进入页拉取预览；**已保存**时展示说明条 + 提供商/主机/模型/Base URL；密钥占位 **`pass.keyPlaceholderSaved`**；**密钥为空且已有密钥**时主按钮为 **「继续向导」**（只更新性格等并跳转），填新密钥则仍走校验与 **`cmd_save_secret`**。
- **`web/src/locales/strings.ts`**：`pass.savedBanner`、`pass.savedProviderLine`、`pass.continueCta` 等。

## 4. 嵌入式 Python：依赖校验 + 清 `PYTHONPATH`

- **`python/src/desktop_entrypoint.py`**：在 **`_wire_sys_path`** 之后 **`_verify_bundle_deps`**：检查 **`site-packages/yaml`**、**`yaml.safe_load`**、**`fastapi`/`uvicorn`**；失败时 **明确日志** 并 **`SystemExit(4)`**，指向 **`build_bundle.ps1`**。
- **`tauri/src/python_supervisor.rs`**、**`tauri/src/weixin_qr.rs`**：**`env_remove("PYTHONPATH")`**，避免宿主 shell 的 **`PYTHONPATH`** 指到 git 树等导致 **错误/残缺 `yaml`** 或缺 **`fastapi`**。

## 5. `python/build_bundle.ps1`：pip 校验、PowerShell here-string、**`site-packages` 删除失败**

### 5.1 pip 安装后校验

- 在 **`pip install`** 之后运行 **`python/tools/verify_bundle_site_packages.py`**（由 **`build_bundle.ps1`** 调用 bundled **`python.exe`**），校验 **`yaml.safe_load`**、**`fastapi`**、**`uvicorn`**。
- **不再**用 **`python -c` + PowerShell here-string** 内嵌多行代码：在 Windows 上易被引号/换行截断，曾反复出现 **`SyntaxError`**。

### 5.2 `pip install`

- 使用 **`--upgrade`**，并先 **`Remove-Item`**（或等价）清空 **`site-packages`**，减轻 **「Target directory … already exists」**。

### 5.3 Windows 文件锁（`.pyd` 访问被拒绝）

- 新增 **`Clear-BundleSitePackages`**：**最多 6 次**间隔 **2 秒**重试删除；仍失败则尝试将 **`site-packages`** **重命名**为 **`site-packages.stale_<unix_ts>`** 再新建空目录继续安装；若仍失败则 **抛出可读错误**（提示关闭 HermesDesk / 结束占用该 runtime 的 **`python.exe`**）。

## 7. 使用与排错提示

1. **必须先**在仓库根目录执行 **`.\python\build_bundle.ps1`**，保证 **`python/dist/runtime/site-packages`** 含 **PyYAML、fastapi、uvicorn** 等；再 **`cargo run`** / 打安装包。
2. **bundle 前**若删不掉 **`site-packages`**：退出 HermesDesk，任务管理器结束 **`python\dist\runtime\python\python.exe`**，必要时用脚本提供的 **rename** 残留目录后再删。
3. **个人微信扫码**：腾讯网页可能无成功提示属正常；以壳内进度/结果 JSON 为准。
